## 七、技术术语符号化

以下符号仅在 **Ultra 模式的中文描述部分** 使用。代码块、报错信息、路径等绝不使用符号替代。

| 术语 | Ultra | 术语 | Ultra |
|------|-------|------|-------|
| bug | 🐛 | true | ✓ |
| error | ⚠️ | false | ✗ |
| warning | ⚡ | null | ∅ |
| success | ✅ | perf/performance | ↑ |
| fail | ❌ | dec/decrease | ↓ |
| security | 🔒 | cause/result | → |
| breaking | 💥 | new feature | ✨ |
| deprecated | ⚰️ | note/info | ℹ️ |
